# v0.1.0 Release

Status: validated - 2026-03-28

## Positioning

`v0.1.0` is the first named baseline of the project.
It is meant to freeze a coherent working state, not to claim production maturity.

## Included in v0.1.0

- Raspberry Pi camera capture loop with staging and persisted captures.
- Motion event detection and SQLite persistence.
- YOLO-based bird detection integrated into the main pipeline.
- Baseline individual recognition using handcrafted features and cosine matching.
- Flask web dashboard with latest image, sightings, event gallery, and live camera mode.
- Camera/service monitoring endpoints and dashboard timeline/highlights.
- Optional webhook alert test endpoint and daily CSV export endpoint.
- Deployment starter kit with `systemd` services, watchdog timer, and `logrotate` config.
- Initial test coverage for matching, features, and API integration points.

## Explicitly Not Claimed by v0.1.0

- Long-duration production stability.
- Fully calibrated individual recognition accuracy.
- Finalized operations playbook for unattended deployment.
- Broad multi-user validation.

First public baseline of Pi5 Bird Feeder. This version freezes a coherent working state around camera capture, motion-triggered detection, baseline individual recognition, dashboard visualization, and deployment scaffolding. It is suitable as a foundation for further field validation and iterative improvement, not yet as a production-stable milestone.
